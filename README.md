# prop-it
[![Build Status](https://travis-ci.org/venkatperi/prop-it.svg?branch=master)](https://travis-ci.org/venkatperi/prop-it)

Attach object properties with:
* custom getters / setters
* backing store
* change events
* readonly
* setter conversion function

## Installation

Install with npm

```shell
npm install prop-it
```
